# SDAC Bootstrap Classwork

## Project Overview
This repository contains the **basic Bootstrap classwork** completed during the SDAC. The purpose of this project is to **practice and understand the core features of Bootstrap**, including:

- Grid system
- Navbar
- Carousel
- Cards
- Buttons
- Forms
- Footer design

---


## Features Implemented

1. **Responsive Navbar**
   - Logo on the left
   - Navigation links aligned right
   - Login button on the right
   - Dark theme with readable text

2. **Image Carousel**
   - Five images with consistent height
   - Responsive, aesthetic design
   - Slide controls (previous/next)
   - Indicators for navigation

3. **Product Cards**
   - Three cards side by side using Bootstrap grid
   - Each card includes an image, title, description, and "Show More" button
   - Centered text, shadow, and rounded corners for modern look
   - Responsive design (cards stack on small screens)

4. **Footer**
   - Dark theme
   - White centered text
