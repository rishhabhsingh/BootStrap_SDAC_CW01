# SDAC Bootstrap Classwork

## Project Overview
This repository contains the **basic Bootstrap classwork** completed during the SDAC. The purpose of this project is to **practice and understand the core features of Bootstrap**, including:

- Grid system
- Navbar
- Carousel
- Cards
- Images
- Buttons
- Forms
- Maps
- Table 
- Drop-down 
- Footer design
- Responsiveness
